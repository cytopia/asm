# print-asm

Print pre-defined sized length message to stdout.


## Requirements

```bash
sudo apt-get install nasm make
```

## Build
```bash
# This
make

# Or this
make build
```

## Run
```bash
$ ./build/myprint
Hello World!
$ echo $?
0
```

## Clean
```bash
make clean
```
